# BLAC'S TECH HUB | Receipt Pro 🚀

A professional, high-performance **Digital Receipt & Business Management System** built for small to medium-sized businesses. This tool handles everything from generating branded receipts to tracking long-term financial growth.

## ✨ Key Features
- **Live Sales Analytics:** Visual 7-day revenue charts using Chart.js.
- **Dynamic Branding:** Fully customizable business name and admin security via the Settings panel.
- **Financial Tracking:** Real-time calculation of "Today's Revenue" vs. "Monthly Totals."
- **Professional Reporting:** One-click PDF report generation for accounting.
- **Secure Access:** Private Admin Terminal with database-driven authentication.

## 🛠️ Tech Stack
- **Backend:** PHP 8.x
- **Database:** MySQL
- **Frontend:** HTML5, Modern CSS3 (Flexbox/Grid)
- **Data Viz:** Chart.js
- **Icons/UI:** Custom branding for BLAC'S TECH HUB

## 🚀 How to Install
1. Clone the repository to your local `htdocs` folder.
2. Import the provided SQL file into your MySQL database.
3. Configure your database credentials in `db_config_sample.php` and rename it to `db_config.php`.
4. Access the dashboard via `localhost/receipt_app/login.php`.
